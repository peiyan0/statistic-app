How to Use

  • Descriptive Statistics:
    • Enter comma-separated numbers or select a sample dataset
    • View mean, median, mode, range, variance, standard deviation, quartiles, and outliers
  • Linear Regression:
    • Enter X and Y values as comma-separated lists
    • View regression equation, RÂ², and correlation coefficient
  • ANOVA:
    • Enter data in format: "group1:1,2,3;group2:4,5,6"
    • View ANOVA table with F-test, degrees of freedom, and sum of squares
  • Confidence Intervals:
      Choose between three types:
    • Two Population CI (enter sample sizes, means, std devs)
    • Dependent Data T-Test (enter before/after measurements)
    • Two Sample Proportion (enter sample sizes and proportions)
  • Datasets:
    • Browse all available sample datasets
    • Click to use in calculations
  • History:
    • View all previous calculations with timestamps

For more detailed instructions, please refer to the documentation or contact support.

Happy calculating!
